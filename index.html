📖 How to use this app

1. Upload files — Drop your .rtf or .txt flight details file and the hotel rooming list (.csv) into the upload boxes above.

2. Generate — Click Generate Manifest. The app matches clients by Reservation ID (e.g. 3070384), falling back to token-based name matching if the ID is missing or unmatched.

3. Read the flags:
EARLY ARR — Client's flight arrives before their hotel check-in date
DATE MISMATCH — Flight arrival date doesn't match check-in (and is not flagged as early)
NO ROOM MATCH — No rooming list entry could be found for this passenger

Tour code → Default hotel:
KTZ → Eka Hotel
UNK → Tamarind Tree Hotel
All others → taken directly from the rooming list

Export — Use Export CSV to download a spreadsheet, or Print for a printer-friendly version.

Note on XLS files: Open your rooming list in Excel, choose Save As → CSV, then upload the .csv here.
